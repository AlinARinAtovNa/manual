## Как создать Pull Request на GitHub (PR)

**Pull Request** — это предложение "влить" вашу ветку `task-1-add-index` в основную ветку `main`.

### Пошаговая инструкция:

| Шаг | Действие |
|-----|----------|
| 1 | После `git push` откройте репозиторий на GitHub.com |
| 2 | Нажмите на баннер **"Compare & pull request"** (или вкладку **Pull requests** → кнопку **New pull request**) |
| 3 | Убедитесь, что выбрано: **base: main** ← **compare: task-1-add-index** |
| 4 | Заполните заголовок PR (например, *"Добавлен index.html"*) |
| 5 | Напишите описание изменений |
| 6 | Нажмите **Create pull request** |
