## Основные команды Git

| Команда | Что делает |
|---------|-------------|
| `git status` | Показать измененные файлы |
| `git add .` | Добавить все файлы в коммит |
| `git commit -m "текст"` | Сохранить версию |
| `git push` | Отправить на GitHub |
| `git checkout -b новая-ветка` | Создать и переключиться на ветку |
| `git checkout main` | Переключиться на ветку main |
| `git log --oneline` | Показать историю коммитов |
